# SJTUFlow 文档索引

SJTUFlow 现在定位为纯本地 Web 应用：用户启动本机后端，浏览器打开前端界面，首次填入模型、Canvas 和资料目录等配置，然后在本地完成 briefing、对话、资料读取、转写和 skill 工作流。
